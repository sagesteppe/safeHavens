# Omernik level 4 ecoregions of California and Oregon

A vector data set of the Omernik level 4 ecoregions clipped to
California and Oregon. Downloaded from
https://www.epa.gov/eco-research/level-iii-and-iv-ecoregions-continental-united-states
and simplified using mapshaper.

A vector data set of Bioregions developed by Morrone et al. 2022
Downloaded from https://neotropicalmap.atlasbiogeografico.com/ and
simplified using mapshaper.

## Usage

``` r
data(WesternEcoregions)
```

## Format

A spatial vector data set.

A spatial vector data set.

## Details

- L4_KEY. Full name and codes of Level 4 ecoregions

- US_L4CODE. Codes of level 4 ecoregions

- US_L4NAME. Names of level 4 ecoregions

&nbsp;

- Provincias Full name of the bioregion
